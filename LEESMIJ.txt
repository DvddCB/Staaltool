RECEPTIE PDF ACCESS V1

Deze versie:
- Receptie tekst onder Orders verwerken klopt nu.
- Receptie kan geen pickbon openen.
- Receptie ziet alleen PDF knoppen als pakbon_url/stickerbon_url gevuld zijn.
- Admin ziet PDF knoppen ook.
- Gebruiker/worker ziet PDF knoppen niet.

Controle:
1. Log in als receptie.
2. Klik een order aan.
3. Bij geselecteerde order verschijnen:
   - Pakbon openen
   - Stickerbon openen
   alleen als de watcher deze links al heeft gekoppeld.

Gebruik:
1. Vervang src/App.jsx
2. Commit changes
3. Wacht op Vercel Ready
4. Ctrl + F5
