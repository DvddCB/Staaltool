App Auth V5 - Wachtwoord vergeten

Nieuw:
- Login-scherm heeft nu knop: Wachtwoord vergeten?
- Gebruiker vult e-mailadres in en klikt op Wachtwoord vergeten?
- Supabase stuurt herstelmail.
- De herstel-link gebruikt de huidige site als redirect.

Belangrijk in Supabase:
Authentication -> URL Configuration
Site URL: jouw live site
Redirect URLs: jouw live site met /*

Gebruik:
1. Vervang src/App.jsx volledig.
2. Commit changes.
3. Wacht op Vercel Ready.
4. Test met een bestaand Supabase e-mailadres.
